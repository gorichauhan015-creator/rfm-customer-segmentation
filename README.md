# RFM Customer Analysis Project

## About the Project

In this project, I worked on a sales dataset and performed customer segmentation using RFM analysis (Recency, Frequency, Monetary). The main aim was to understand customer behavior and identify different types of customers based on their purchasing patterns.

---

## Tools Used

* PostgreSQL for data cleaning and analysis
* Power BI for dashboard and visualization
* Excel for handling data files

---

## What I Did

* Cleaned the raw dataset using SQL (handled null values, duplicates, etc.)
* Created RFM metrics:

  * Recency (last purchase)
  * Frequency (number of purchases)
  * Monetary (total spend)
* Segmented customers into groups like:

  * Champions
  * Loyal Customers
  * At Risk
  * Lost Customers
* Built an interactive dashboard in Power BI to visualize the results

---

## Key Learnings

* Understood how customer segmentation works in real business scenarios
* Learned how to clean and prepare messy data
* Improved my SQL and Power BI skills
* Got hands-on experience in building dashboards
